# Learning React

This repository contains my React learning journey with 8 different projects, each focusing on different concepts and features of React.

## Projects Overview

### 1. 01basicreact
- Basic React setup and fundamentals
- Introduction to JSX and components

### 2. 01vitereact
- React with Vite build tool
- Modern development setup

### 3. 02counter
- State management with useState
- Event handling and counter functionality

### 4. 03tailwindprops
- Tailwind CSS integration
- Props and component communication

### 5. 04bgChanger
- Background color changer application
- State management and event handling
- Tailwind CSS styling

### 6. 05passwordgenerator
- Password generator application
- useCallback and useEffect hooks
- Clipboard API integration

### 7. 06currency
- Currency converter application
- Custom hooks and API integration

### 8. 07reactRouter
- React Router implementation
- Navigation and routing
- Layout components

### 9. 08miniContext
- Context API implementation
- State management across components
- Login and profile functionality

## Technologies Used
- React 19+
- Vite
- Tailwind CSS
- React Router
- Context API

## Getting Started

Each project can be run independently. Navigate to any project folder and run:

```bash
npm install
npm run dev
```
